# 06.3 — Herramientas Recomendadas

---

## 🛠️ Introducción

Este documento recoge herramientas recomendadas para la investigación forense y periodística. Está organizado por categorías y tipo de uso.

**Nota:** Algunas herramientas pueden requerir formación específica. Investiga antes de usar.

## 💻 HERRAMIENTAS DIGITALES

### Investigación OSINT (Open Source Intelligence)

| Herramienta | Descripción | Uso Recomendado |
|-------------|-------------|-----------------|
| **Maltego** | Herramienta de análisis de relaciones y conexiones. | Identificar redes de contactos. |
| **Shodan** | Motor de búsqueda de dispositivos conectados a internet. | Investigación de infraestructura digital. |
| **Google Dorks** | Técnicas de búsqueda avanzada en Google. | Encontrar información oculta o no indexada. |
| **Wayback Machine** | Archivo histórico de páginas web. | Ver contenido eliminado o modificado. |
| **SpiderFoot** | Herramienta de recopilación de información OSINT. | Reconocimiento digital del sujeto. |

| **TweetDeck** | Monitorización de Twitter en tiempo real. | Seguimiento de actividad. |
| **Instagram-scraper** | Herramienta para extraer datos públicos de Instagram. | Análisis de perfiles. |

### Verificación de Datos
